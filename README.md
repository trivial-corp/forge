# Trip1 Forge

Free online tools by [Trip1](https://trip1.com/?utm_source=forge&utm_medium=readme&utm_campaign=forge). No sign-up, no install, runs entirely in your browser.

**[Open Forge](https://trivial-corp.github.io/forge/)**

## Tools

### Image Tools
Photo editor with watermark removal, background removal, and mask painting. Features brush and magic wand tools, auto background selection, and an inpainting engine.

### Email Signature Generator
Create professional HTML email signatures. Choose from Classic, Compact, or Minimal layouts. Copy as rich text or raw HTML, ready to paste into any email client.

## Development

Each tool is a single self-contained HTML file with no build step. Just open any `.html` file in a browser or push to `main` to deploy via GitHub Pages.
